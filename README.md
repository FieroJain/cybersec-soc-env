# CyberSec-SOC-OpenEnv

A reinforcement learning environment simulating a Security Operations Center (SOC) defending enterprise networks from cyber attacks.

## Live Environment
https://Fieerawe-cybersec-soc-env.hf.space

## Features
- Procedural enterprise network generation
- Multi-stage cyber attack simulation
- Partial observability
- OpenEnv compatible API
- HuggingFace Spaces deployment

## API
POST /reset  
POST /step  
GET /tasks
